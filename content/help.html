Birthdays

There seems to be a problem with entering birthday data in some versions of Zarafa Webaccess. It does work in the Webapp.
This is why there is a check box "Send Birthday Information" in the account settings dialog.
This is a bug in Zarafa not in TbSync.

Debug

Set tbsync.debugwbxml to true in about:config to print the xml being sent and received by TbSync in the error console. It will also dump the raw wbxml in [TB profile folder]/TbSync/wbxml-debug.log.

Switch off debug output if no longer needed and delete the log file.
