Birthdays

There seems to be a problem with entering birthday data in some versions of Zarafa Webaccess. It does work in the Webapp.
This is why there is a check box "Send Birthday Information"
This is a Zarafa bug not TbSync

gcontactsync

gcontactsync add its own aniversary field as does TbSync.
You can disable showing the anniversary field in TbSync by setting tbsync.showanniversary to false in about:config.

Extra ActiveSync Fields

TbSync will add in extra fields for phone numbers in Contact main tab.
To hide them set tbsync.hidephones to true in about:config.
You may need this if you use other addons that alter the Contact display.

Debug

Set tbsync.debugwbxml to 1 in about:config

This will show you the xml being sent and received by TbSync in the error console.
It will also dump the raw wbxml in profile/wbxml.
Turn off after use, and delete profile/wbxml/all the stuff here
